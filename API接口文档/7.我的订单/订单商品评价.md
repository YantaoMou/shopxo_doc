#### 请求URL

> order/commentssave

#### 请求方式

> POST

#### 公共参数

| 参数名 | 是否必须 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- | --- |
| application | 是 | string | web | 请求应用、参考公共中详解 |
| application\_client\_type | 是 | string | pc | 请求客户端、参考公共中详解 |
| token | 否 | string |  | token |

#### 请求参数

```
{
    "is_anonymous": "0",
    "id": "113",
    "goods_id": "[12]",
    "rating": "[5]",
    "content": "["宝贝非常好！"]",
    "images": "[{"https://xxx/1633853346787502.png"},{"https://xxx/1633853346688315.png"}]"
}
```

| 参数名 | 是否必须 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- | --- |
| id | 是 | int |  | 订单id |
| is\_anonymous | 是 | int |  | 是否匿名（0否、1是） |
| goods\_id | 是 | int |  | 商品id |
| rating | 是 | int |  | 评分 |
| content | 是 | int |  | 评价内容 |
| images | 是 | int |  | 评价图片 |

#### 返回示例

**正确时返回**

```
{
    "msg": "评论成功",
    "code": 0,
    "data": ""
}
```

**错误时返回**

```
{
    "msg": "error",
    "code": -1,
    "data": []
}
```

#### 返回参数说明 -> data

无
